# Feature

This chapter describe PyPSA-ASEAN features that does not exist in the OS PyPSA-Earth. 

- [Append Cost Data](append-cost-data.md)
- [ASEAN Grid Infrastructure](asean-grid-infrastructure.md)
- [Final Adjustment](final-adjustment.md)
- [Dual Pull Request](dual-pull-request.md)

While PyPSA-Earth provides the foundational framework for modeling the ASEAN region, it requires additional tailoring to capture unique regional characteristics, such as the ASEAN Power Grid. PyPSA-ASEAN bridges this gap by allowing users to overwrite default datasets with more robust, regional statistics from the ASEAN Centre for Energy (ACE) and the IEA. Furthermore, for features with global utility, a "dual pull request" strategy is employed to ensure that enhancements are integrated into both the regional and global versions of the framework.
