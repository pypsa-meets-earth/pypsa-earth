# Electricity data sources

## Small-scale solar installation postcode data (CER)

Source:
https://cer.gov.au/markets/reports-and-data/small-scale-installation-postcode-data

File:
sgu-solar-capacity-2011-to-present-and-totals.csv

License:
Not specified on the source page; reuse conditions should be checked against the Clean Energy Regulator website copyright notice.

Recommended citation:
Clean Energy Regulator, Small-scale installation postcode data, sgu-solar-capacity-2011-to-present-and-totals.csv, accessed April 2026.

Notes:
- This dataset contains small-scale solar installation capacity by postcode.
- The column "Total Rated Power Output In kW" provides cumulative installed capacity up to the latest month included in the file.
- For a 2025 base year, installed rooftop solar capacity should be computed cumulatively up to and including Dec 2025.
- This dataset is used to estimate existing rooftop PV capacity, separately from utility-scale solar plants in powerplants datasets.
